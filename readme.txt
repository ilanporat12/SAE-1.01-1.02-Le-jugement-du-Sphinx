Le jugement du sphinx est un jeu éducatif destiné aux élèves de niveau collège.
Le jeu teste les connaissances de l’élève sur 3 différent thèmes : l’histoire, SVT et physique chimie avec des questions pouvant varier en difficulté si le joueur enchaîne les bonnes réponses. Le déroulement de la partie est simple, le sphinx nous pose une question et nous devons répondre (attention à l’orthographe des réponses). Les conditions pour gagner sont de répondre correctement à 15 questions, cependant le joueur perd s’il répond faux à 3 questions. Un mode 2 joueurs est disponible où le joueur ayant le plus grand score gagne.

Pour fonctionner, le jeu nécessite le langage de programmation Ijava.
Pour le lancer :
1)	Ouvrez un terminal
2)	Placez-vous dans le dossier « JugementDuSphinx »
3)	Exécuter les commandes suivantes :
		ijava start
		ijava execute jeux
4)	Suivez les instructions à l’écran, bon jeu !
